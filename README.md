# Warehouse Management System

Консольная система управления складом, разработанная на языке Python.

## Описание проекта

Warehouse Management System — это приложение для управления складом, позволяющее выполнять основные операции с товарами: добавление, удаление, редактирование, поиск и просмотр статистики. Проект разработан в учебных целях для закрепления навыков программирования на Python.

## Функциональные возможности

- Просмотр списка товаров
- Добавление нового товара
- Удаление товара
- Редактирование информации о товаре
- Поиск товара по различным параметрам
- Продажа товара
- Пополнение количества товара
- Просмотр категорий товаров
- Просмотр статистики склада
